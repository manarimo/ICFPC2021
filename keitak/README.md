ランダムサブセットサムジェネレーター

src/sticks.tsがコード

実行は
`yarn tsc -w`
でJS生成
`node dist/sticks.js`
で実行

X軸は、整数をEDGE_MINからEDGE_MAXまでランダムに生成。（最後のあまりはその範囲内とは限らない）
その整数を使って、＋ーのサブセットサム

Y軸も基本的に同じだけど、サブセットサムの過程で、座標が絶対に０未満にならず、最後の座標が最大座標になる。
これは、そうしないと４つの角をカバーできないから。

WIDTH,HEIGHTが元になるFigureの四角形の幅と高さ。
今の所頂点数を正確に指定はできない。
